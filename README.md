# 🚀 Mentor Bridge (`Mymentor.in`)

> **Status:** 🚧 *In Active Development*

**Mentor Bridge** is an upcoming peer-to-peer student mentorship web platform designed to foster meaningful student-to-student guidance, identity verification, and community engagement.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Version Control & Hosting:** Git, GitHub, GitHub Pages
- **Developer Tools:** VS Code, Live Server

---

## ✨ Current Features (In Progress)

- [x] **Student Identity Verification Flow:** Responsive verification form featuring custom client-side validation, regex email domain filtering, and real-time input error handling.
- [x] **UI/UX Polish:** Non-blocking floating toast notifications and custom error styling without browser alert interruptions.
- [x] **Responsive Layout:** Optimized card and container structures for seamless desktop and mobile viewports.

---

## 📅 Roadmap / Next Steps

- [ ] Connect interactive community & chat screens.
- [ ] Implement local state management for student profile data.
- [ ] Refine navigation transitions across all subpages.
